Syntax reference

Simple.
Powerful. Yours.

Every Leo program follows a clean, readable structure.

Program structure

@leo // start // your code @leo.stop() // end

Variables

leo x == 10 leo name == ask(("prompt")) leo s == ("hello")

Arithmetic

a <+> b // addition a (-) b // subtraction a <*> b // multiply a </> b // division

Text styling

("hi")-(a1) // small ("hi")-(a3) // large ("hi")-(c-type==blue) // color

Output

preview(x) preview(("hello")-(a2))

Repeat loop

/repeat(3) { leo x == ("leo!") } preview(x)
